## A sublime pulgin for newing a file and adding a jekyll post head

### Install

- open your sublime 
- Tools -> New Plugin
- copy the .py file into the new plugin file and save
- copy the keymap and save
- done.

### Use It

- OSX:command+shift+c

``` javascript
	---
	layout: post 
	title:  
	date: 2014-11-23 08:34:03 
	category:  
	---
```
